# Algoritmo de planificación por turnos (Round Robin)

En este proyecto se implementa el algoritmo de planificación de
procesos en los siguientes lenguajes:

* Ruby
* Python
* Java
* Crystal

# Ejemplo

* Cantidad de Procesos: 2000
* Instrucciones: 100
* Quantum: 30
